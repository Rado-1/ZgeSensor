ZGE demo project #1 - Demonstrates usage of accelerometer and light sensors in ZGameEditor application.

ZGE demo project #2 - Demonstrates usage of rotation vector sensor in ZGameEditor application. This is a variant of the RotationVectorDemo from Android SDK samples implemented in ZGE.

ZGE demo project #3 - Demonstrates usage of rotation vector sensor to move player in a 3D space. Rotate the screen to rotate the camera view, touch right side of the screen with a single finger to move forward, left side of the screen to move backward, and touch the screen with three fingers to regenerate the scene and reset player's position.
